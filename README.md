# PostPerfect - Monthly Canva Templates

This repo contains the sales page and product files for PostPerfect, a monthly Canva template subscription sold on Ko-fi.

## What This Product Is

Monthly social media template kits for small business owners. Subscribers get 15+ customizable Canva templates, a content calendar and strategy guides each month for $17.

## What's In This Repo

- Sales page copy
- Customer welcome message
- Product descriptions
- Template links and access files

## How It Works

1. Customer subscribes on Ko-fi ($17/month)
2. They automatically receive a text file via email
3. Text file contains Canva template links and instructions
4. Templates work with free Canva accounts

Built for entrepreneurs who need professional social media content without spending hours designing.
